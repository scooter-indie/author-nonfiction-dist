Thank you for your interest in contributing!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Reporting Issues

- Use the [GitHub Issues](https://github.com/scooter-indie/author-nonfiction/issues) page
- Check existing issues before creating a new one
- Provide clear steps to reproduce bugs
- Include framework version and environment details

## Development Setup

1. Clone the repository
2. The framework is primarily Markdown-based with no build step required
3. Test changes with a sample book project

## Code of Conduct

Be respectful and constructive in all interactions.
