# AI-Assisted Nonfiction Authoring Framework

**This is the distribution repository.**

Version: v0.14.1

## Quick Start

1. Clone this repository to start your book project
2. Read `INSTALLATION.md` for setup instructions
3. Run `configure.md` to initialize your project

## For Claude Code Users

After cloning, run in Claude Code:
```
/fw-init
```

Then follow `configure.md` to set up your book project.

## For Claude Desktop Users

1. Upload `system-instructions.md` to Project knowledge (use the **Add from computer** button; do not paste)
2. Copy the contents of `configure.md` and paste into the chat
3. Follow the interactive prompts to set up your project
